# 설치 절차

1.  Wsl 설치

    - 인터넷 검색에서 wsl 을 검색하면 mocrosoft 사의 wsl 설치 패이지를 찾을 수 있음. 해당 페이지를 참고해서 wsl 설치 진행

    - Terminal을 admin 권한으로 실행한 후, 'wsl --install' 실행. 설치 후 윈도우 재시작 필요

    - wsl --install 이라는 기본 명령어를 주었기 때문에 리눅스 배포판이 기본판이 설치됨. 확인은 설치된 후에 terminal 창이 뜸

    - 최신 우분투 22.04.1 LTS 버전이 설치됨

    - wsl --list --online : 사용가능한 배포판 목록 확인 centos는 없음 =\> 뒤에서 별도 설치 방법 설명

    - wsl --install -d \<DistributionName\> : 설치할 배포판을 지정해서 설치 (최초 설치할 때 이런 방식으로 설치할 수도 있었음)

2.  wsl 버전 확인

- terminal 이나 powershell 에서 'wsl -l -v' 명령어 입력하면 wsl 의 버전이 나옴

- wsl 명령과 함께 사용하는 기본 Linux 배포판을 설정하려면 'wsl -s \<DistributionName\>' 또는 'wsl --setdefault \<DistributionName\>을 입력

- 'wsl --set-version' 명령으로 WSL 2를 WSL 1로 다운그레이드하거나 업데이트 가능

- 버전 변경은 'wsl --set-version \<distributionName\> 2' 와 같은 형식으로 사용

## wsl 명령어

wsl ~ : 디렉토리 홈으로 변경\
wsl --set-default \<DistributionName\> : 기본 Linux 배포판 설정\
wsl --distribution \<distributionName\> --user \<User Name\> : 해당 배포판 실행하기\
wsl --update : wsl 버전을 최신 버전으로 업데이트\
wsl --status : 상태\
wsl --version : 버전\
wsl -u \<username\>`, `wsl --user \<username\> : 특정 사용자로 실행\
\<distributionName\> config --default-user \<Username\> : 배포의 기본 사용자 변경\
wsl --shutdown : 가상 머신 즉시 종료\
wsl --terminate \<DistributionName\> : 지정된 배포판 종료 및 실행 중지\
wsl --export \<DistributionName\> \<FileName\> : 지정된 tar 파일을 새 배포로 가져오고 내보냄\
wsl --import \<distributionName\> \<InstallLocation\> \<FileName\>\
wsl --unregister \<DistributionName\> : 배포판 등록 취소 또는 제거

사례 : Error 0x80040326 발생시
