- \[System.Environment\]::GetEnvironmentVariable('FOO', ‘bar’, \[System.EnvironmentVariableTarget\]::Machine)

#### 시스템 환경 변수 삭제

- \[System.Environment\]::SetEnvironmentVariable('FOO', ‘’, ‘Machine’)

# Windows 여유 공간 확보

1.  Service windows update 정지

2.  C:\windows\SoftwareDistribution 폴더 삭제

3.  C:\windows\system32\catroot2 폴더 삭제

4.  실행 %temp% 로 해당 폴더로 이동하여 삭제 (C:\Users\id\AppData\Local\Temp

    1.  temp 폴더 삭제 실행 파일 생성

        1.  del C:\Users\\username%\AppData\Local\Temp /f /s /q

        2.  rd C:\Users\\username%\AppData\Local\Temp /s /q

        3.  del C:\Windows\Temp /f /s /q

        4.  rd C:\Windows\Temp /s /q

# IIS 잡다한 팁

- IIS process Recycle - appDomain recycle 조건

  1.  Modification to web.config or Global.asax

  2.  change to the contents of the application’s bin directory

  3.  change to the physical path of the virtual directory

  4.  Deletion of the subdirectory of the application

  5.  The number of re-compilations (aspx, ascx or asax) exceeds the limit specified by teh setting in machine.config or web.config (default of 15)
