# JEUS

## container start/stop script

- jeus 7.0 이상

#### start

- jcboot

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  #!/bin/sh
  export JEUS_HOME=/sw/jeus8_5
  export PATH="${PATH}:${JEUS_HOME}/bin:${JEUS_HOME}/webserver/bin:${JEUS_HOME}/lib/system"
  export DOMAIN_NAME=jeus_domain
  export DAS_HOSTNAME="hostname"
  export DAS_PORT=10000
  export DAS_URL=${DAS_HOSTNAME}:${DAS_PORT}
  export ENCODE_FILE=${JEUS_HOME}/bin/jeusEncode

  jeusadmin -host $DAS_URL -domain $DOMAIN_NAME -u wasadmin -cachelogin -f $ENCODE_FILE "start-server $1"
  ```

  </div>

  </div>
