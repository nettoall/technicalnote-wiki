# <span id="title-text"> Tech : Sonarqube 자동 기동시 오류 </span>

</div>

<div id="content" class="view">

<div class="page-metadata">

<div id="main-content" class="wiki-content group">

GCP 클라우드에서 비용 절감을 위해 dev, stage 서버를 업무중에만 사용하도록 매일 08 ~ 21시까지만 사용하도록 정책 설정

08시 VM 재기동시 해당 VM에 설치된 솔루션 및 서버가 자동 기동됨

이 와중에 sonarqube 서버가 정상적으로 기동되지 않는 현상을 발견

원인으로 wrapper.conf에 설정된 wrapper.java.command=/path/to/my/jdk/bin/java

or wrapper.java.command=java가 정상적으로 동작하지 않는 것으로 판단됨

java path를 실 경로로 수정함

wrapper.conf 에 wrapper.java.commond=실제 자바 경로

조치 후 정상 기동 및 서비스됨
