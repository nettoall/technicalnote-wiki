# PC설치

## 환경 구성 및 샘플 설치

1.  VirtualBox 설치

    1.  추천 6.1.12 http://www.virtualbox.org/wiki/Downloads

2.  베이그런트 설치

    1.  https://www.vagrantup.com/downloads.html

    2.  사용자의 요구에 맞게 시스템 자원을 할당, 배치, 배포해 두었다가 필요할 때 시스템을 사용할 수 있는 상태 제공 : 프로비저닝(provisioning)

    3.  여러 대의 가상 머신을 추가, 설정할 수 있는 장점

3.  베이그런트 구성(샘플 가상머신 설치)

    1.  베이그런트 설치 디렉토리(f:\HashiCorp)에 프로비저닝에 필요한 모든 코드 작성

    2.  vagrat init : 베이그런트 초기화 → Vagrantfile 생성됨

    3.  Vagrantfile 내 config.vm.box = “base” 확인

    4.  vagrant up 실행 → error 발생

        1.  config.vm.box 이미지가 base 이나 베이그런트가 해당 이미지를 찾지 못해 에러 발생

        2.  에러가 발생하지 않게 설치할 운영 체제 이미지 선택 필요

        3.  <a href="https://app.vagrantup.com/boxes/search" class="external-link" rel="nofollow">https://app.vagrantup.com/boxes/search</a> 에서 “sysnet4admin” 검색

        4.  “sysnet4admin/CentOS-k8s” 확인

        5.  Vagrantfile 내 config.cm.box=”`sysnet4admin/CentOS-k8s`" 로 변경

    5.  vagrant up 다시 실행 → sysnet4admin/CentOS-k8s 다운받음

        1.  설치 끝에 Vagrant was unable to mount VirtualBox shared folders. 란 내용은 버추얼 박스의 게스트 에디션이 설치되지 않아 발생 → 필요하지 않아 설치 안함

    6.  VirtualBox에 생성되었나 확인 → HashiCorp_default_165945404080 으로 실행중 표시

    7.  vagrant ssh 로 CentOS 접속

        1.  실행시간 확인 : uptime

        2.  운영체제 종류 : cat /etc/redhat-release → CentOS Linux release 7.8.2003(Core)

        3.  exit 로 ssh 종료

    8.  vagrant destroy -f : 설치된 가상 머신 삭제

## 베이그런트로 테스트 환경 구축

### 가상 머신 설치 및 shell 추가

- 베이그런트 코드는 Ruby 로 작성됨

- eth0(NAT 네트워크), eth1(호스트 전용 네트워크) 네트워크 드라이버 2개 설정

- m-k8s 2cpu, 2GB 메모리 설정

1.  Vagrantfile 수정

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    # -*- mode: ruby -*-
    # vi: set ft=ruby :
    Vagrant.configure("2") do |config|
      config.vm.define "m-k8s" do |cfg|
        cfg.vm.box = "sysnet4admin/CentOS-k8s"
        cfg.vm.provider "virtualbox" do |vb|
            vb.name = "m-k8s(github_SysNet4Admin)"
            vb.cpus = 2
            vb.memory = 2048
            vb.customize ["modifyvm", :id, "--groups", "/k8s-SM(github_SysNet4Admin)"]
          end
          cfg.vm.host_name = "m-k8s"
          cfg.vm.network "private_network", ip: "192.168.1.10"
          cfg.vm.network "forwarded_port", guest: 22, host: 60010, auto_correct: true, id: "ssh"
          cfg.vm.synced_folder "../data", "/vagrant", disabled: true
        end
      end
    ```

### 가상머신 여러대 설치

- 기존 설치 가상 머신 외에 3대 추가 설치

1.  Vagrantfile 작성\

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    # -*- mode: ruby -*-
    # vi: set ft=ruby :
    Vagrant.configure("2") do |config|
      config.vm.define "m-k8s" do |cfg|
        cfg.vm.box = "sysnet4admin/CentOS-k8s"
        cfg.vm.provider "virtualbox" do |vb|
            vb.name = "m-k8s(github_SysNet4Admin)"
            vb.cpus = 2
            vb.memory = 2048
            vb.customize ["modifyvm", :id, "--groups", "/k8s-SM(github_SysNet4Admin)"]
          end
          cfg.vm.host_name = "m-k8s"
          cfg.vm.network "private_network", ip: "192.168.1.10"
          cfg.vm.network "forwarded_port", guest: 22, host: 60010, auto_correct: true, id: "ssh"
          cfg.vm.synced_folder "../data", "/vagrant", disabled: true
          cfg.vm.provision "shell", path: "install_pkg.sh" 
          cfg.vm.provision "file", source: "ping_2_nds.sh", destination: "ping_2_nds.sh"
          cfg.vm.provision "shell", path: "config.sh"
        end
      
      #=============#
      # Added Nodes #
      #=============#
      
      (1..3).each do |i| # 1부터 3까지 3개의 인자를 반복해 i로 입력
        config.vm.define "w#{i}-k8s" do |cfg| # {i} 값이 1, 2, 3으로 차례대로 치환됨
          cfg.vm.box = "sysnet4admin/CentOS-k8s"
          cfg.vm.provider "virtualbox" do |vb|
            vb.name = "w#{i}-k8s(github_SysNet4Admin" # {i} 값이 1, 2, 3으로 차례대로 치환됨
            vb.cpus = 1
            vb.memory = 1024
            vb.customize ["modifyvm", :id, "--groups", "/k8s-SM(github_SysNet4Admin"]
          end
          cfg.vm.host_name = "w#{i}-k8s" # {i} 값이 1, 2, 3으로 차례대로 치환됨
          cfg.vm.network "private_network", ip: "192.168.1.10#{i}"
          cfg.vm.network "forwarded_port", guest:22, host: "6010#{i}", auto_correct: true, id: "ssh"
          cfg.vm.synced_folder "../data", "/vagrant", disabled: true
          cfg.vm.provision "shell", path: "install_pkg.sh"
        end
      end
    end
    ```
