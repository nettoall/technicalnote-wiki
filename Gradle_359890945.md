- JVM, Groovy(<a href="https://groovy-lang.org/" class="external-link" data-card-appearance="inline" rel="nofollow">https://groovy-lang.org/</a> )

gradle project 생성

gradle init

gradle wrapper 생성

gradle wrapper

다음 파일이 생성됨

gradlew

gradlew.bat(win)

gradle 폴더 하위에 wrapper 에 gradle-wrapper.properties

distributionUrl=file\\/C:/gradle/gradle-6.9.2-bin.zip

설정해주면 인터넷 연결 안 된 상태에서 로컬에서 파일 가져와서 사용함

- Annotation Processor

  - 컴파일 단계에서 Annotation에 정의된 일렬의 프로세스를 동작하게 하는 것을 의미

  - 컴파일 단계에서 실행되기 때문에 빌드단계에서 에러를 출력하게 할 수 있고 소스코드 및 바이트 코드를 생성할 수 있음

  - lombok의 경우 @Getter, @Setter 등의 annotation이 Annotation Processor를 제공하여 표준적으로 작성해야 할 코드를 개발자 대신 생성

  - 컴파일 시점에 abstract syntaxtree 조장

  - 구조

    - Processor interface를 통하여 Annotation Processor를 구현

    - 여러 rounds를 거쳐 Source 및 Compile된 코드 처리
