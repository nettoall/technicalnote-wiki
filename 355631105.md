# 1. VirtualBox 하에서 실습예제

## 1.1 설치

- <a href="https://www.virtualbox.org" class="external-link" rel="nofollow">https://www.virtualbox.org</a> 에서 다운 로드 후 설치

## 1.2 VirtualBox network 구성

- VirtualBox network 구성

  - 파일 \> 환경설정 \> 네트워크 \> 추가

    - 네트워크 이름 : localNetwork

    - CIDR : 10.100.0.0/24

    - DHCP 지원

    - 포트 포워딩

      - docket1 TCP 127.0.0.1 105 10.100.0.105 22

      - docker2 TCP 127.0.0.1 106 10.100.0.106 22

## 1.3 가상머신 만들기

새로만들기

- docker-ubuntu

  - 이름 : docker-ubuntu

  - 종류 : Linux

  - 버전 : ubuntu(64bit)

  - 메모리 : 2GB

  - 하드디스크 : 10 GB(기본값)

  - 하드디스크 종류 : VDI

  - 물리적 하드 디스크 : 동적할당

  - 파일 위치 및 크기 : 20GB 로 수정

  - 생성 후 환경에서 설정

    - 시스템 마더보드 : 플로피 제외

    - 프로세서 개수 : 2

    - 네트워크 : NAT 네트워크(localNetwork)

- docker-centos

  - 이름 : docker-centos

  - 종류 : Linux

  - 버전 : redhat(64bit)

  - 메모리 : 2GB

  - 하드디스크 : 10 GB(기본값)

  - 하드디스크 종류 : VDI

  - 물리적 하드 디스크 : 동적할당

  - 파일 위치 및 크기 : 20GB 로 수정

  - 생성 후 환경에서 설정

    - 시스템 마더보드 : 플로피 제외

    - 프로세서 개수 : 2

    - 네트워크 : NAT 네트워크(localNetwork)

## 1.4 OS 설치

1.  docker-ubuntu 설치

    1.  <a href="http://ubuntu.com" class="external-link" rel="nofollow">ubuntu.com</a> 20.04 설치 (desktop) 다운로드

    2.  설치시 설정 항목

        - 언어선택(한국어)

        - 키보드 선택

        - 일반설치

        - 새로 만들 디스크 지우고 설치

        - 지금 설치

        - 포맷

        - 계속 설치

        - 타임존 설정

        - 관리자 계정 생성: guru/work

        - 컴퓨터 이름 : docker-ubuntu.example.com

2.  docker-centos

    1.  <a href="http://centos.org" class="external-link" rel="nofollow">centos.org</a> 에서 centos7 다운로드

    2.  설치시 설정 항목

        - 언어 : english

        - DATA & Time : Asia/Seoul

        - SOFTWARE SELECTION : GNOME Desktop

        - INSTALLATION DESTINATION : default

        - NETWORK & HOST NAME

          - Ethernet : On

          - Host name : docker-centos.example.com

          - Configuration

            - Method : Manual

            - Address : 10.100.0.106

            - Netmask : 24

            - Gateway : 10.100.0.1

            - DNS Servers : 10.100.0.1

        - USER SETTING

          - Root password : password (두번 클릭시 적용)

        - USER CREATE

          - guru/work

        - 설치 후 재부팅

        - License 동의

        - finish configuration

## 1.5 OS 설정

1.  docker-ubuntu

    - 설정

      - 디스플레이: 1280 X 960 수정

      - 네트워크 : 유선 설정

        - IPv4 방식 : 수동

        - 주소 : 10.100.0.105

        - 네트마스크 : 24

        - 게이트 웨이 : 10.100.0.1

        - 네임서버 : 10.100.0.1

      - terminal 창

        - ip addr : 10.100.0.5 확인

        - hostname : docker-ubuntu (수정)

        - sudo vi /etc/hostname

          - <a href="http://docker-ubuntu.example.com" class="external-link" rel="nofollow">docker-ubuntu.example.com</a>

        - sudo vi /etc/hosts

          - 127.0.1.1 <a href="http://docker-ubuntu.example.com" class="external-link" rel="nofollow">docker-ubuntu.example.com</a> 삭제

          - 10.100.0.105 <a href="http://docker-ubuntu.example.com" class="external-link" rel="nofollow">docker-ubuntu.example.com</a> docker-ubuntu 추가

          - 10.100.0.106 <a href="http://docker-centos.example.com" class="external-link" rel="nofollow">docker-centos.example.com</a> docker-centos 추가

        - ping -c 3 8.8.8.8 (구글 dns 로 ping) → 네트워크 연결 상태 확인

      - root 패스워드 설정

        - sudo passwd root

        - password 로 수정

      - Text 부팅 지원(root)

        - ubuntu desktop 으로 설치를 하여 부팅시 GUI로 시작됨

        - systemctl set-default multi-user.target

      - SSH 서버 설치 (root)

        - apt-get update

        - apt-get install -y openssh-server curl vim tree (openssh-server, curl, vim, tree 설치)

      - sshd 방화벽 오픈

        - sudo ufw allow 22/tcp

      - 설치 확인

        - systemctl status sshd

        - ssh guru@localhost

      - 초기 설치시 한글로 설치할 시 이후 터미날에서 다이아몬드 표시 같이 한글 깨짐이 나옴

        - /etc/default/locale 에서 en_US.UTF-8 로 바꿈

      - 

2.  docker-centos

    - 상위 오른쪽 부분에서 마우스 팝업에서 설정 선택

    - devices \> display 에서 1400 X 1050 선택

    - Region & Language 에서 한국어를 선택

    - 바탕화면에서 마우스 팝업 메뉴에서 terminal 선택

    - ip addr 확인

    - gnome 으로 선택하면 네트워크가 가상화버전으로 실행됨(virbr0 생성됨)

    - systemctl stop libvirtd

    - systemctl disable libvertd

    - /etc/sudoers 파일에 guru 계정 추가하기

    - vi /etc/hostname 확인

    - vi /etc/hosts 에서 docker 정보 입력

      - 10.100.0.105 <a href="http://docker-ubunbu.example.com" class="external-link" rel="nofollow">docker-ubuntu.example.com</a> docker-ubuntu

      - 10.100.0.106 <a href="http://docker-centos.example.com" class="external-link" rel="nofollow">docker-centos.example.com</a> docker-centos

    - 외부 접속 확인

      - ping 8.8.8.8 -c 3

    - systemctl set-default multi-user.target (text 로그인)

    - sshd 기본 설정

      - centos8 은 sshd가 설치가 안 되어 있어 설치 필요

      - sudo yum install openssh-server openssh-clients openssh-askpass

      - /etc/sshd/sshd_config 에서 Port 번호 지정, PermitRootLogin no 로 설정

      - firewall-cmd --list-all 로 방화벽 목록 조회

      - firewall-cmd --permanent --zone=public --add-port=포트번호/tcp 로 추가

      - firewall-cmd --reload : 적용한 방화벽 바로 시작

    - yum install tree

3.  ssh 프로그램으로 원격 접속

    - 1.2 에서 설정한 포트 포워딩 정보로 ssh 프로그램으로 접속 가능함

      - 127.0.0.1:105 로 docker-ubuntu 접속

      - 127.0.0.1:106 로 docker-centos 접속

    - 기타 정보

      - cat /etc/os-release : OS 정보

      - free -h : 메모리 정보

      - uname -r : 커널 정보
