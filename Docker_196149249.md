# 정의

- 반가상화보다 좀더 경량화된 방식

-  lxc(Linux Container) 드라이버 기반으로 동작했지만 0.9x 버전부터는 <a href="https://github.com/docker/libcontainer" class="external-link" rel="nofollow">libcontainer</a>를 사용

- 이미지 버전 관리 기능을 제공

- 이미지는 필요한 프로그램과 라이브러리, 소스를 설치한 뒤 파일로 만든 것입니다. 이 이미지를 저장소에 올리고, 받을 수 있습니다.

- 컨테이너는 이미지를 실행한 상태

- 이미지로 여러개의 컨테이너를 만들 수 있음

- Docker는 특정 실행파일 또는 스크립트를 위한 실행 환경
