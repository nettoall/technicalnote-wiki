# 설정

## 사용자 생성

- Windows 에서 “SQL SHELL(psql)” 실행

- 별도의 입력이 없으면 default 설정으로 진행됨

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  Server[localhost]:
  Database [postgres]:
  Port [5432]:
  Username [postgres]:
  postgres 사용자 암호:
  ```

  </div>

  </div>

- DB 접속 후 사용자 생성

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  postgres=# CREATE ROLE pythonusr WITH LOGIN PASSWORD 'dlatl00'
  ```

  </div>

  </div>

- database 생성

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  postgres=# CREATE DATABASE pythondb WITH ENCODING='utf8' OWNER=pythonuser
  postgres=# GRANT ALL PRIVILEGES ON DATABASE pyhondb to pythonuser
  ```

  </div>

  </div>

- 데이터 베이스 drop

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  drop databae pythondb;
  drop database IF EXISTS pythondb
  ```

  </div>

  </div>

- 데이터 베이스 확인 : \1

- 사용자 조회 : \du;

- 사용자 삭제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  DROP OWNER BY <username>
  DROP ROLE <username>
  DROP USER <username>
  ```

  </div>

  </div>
