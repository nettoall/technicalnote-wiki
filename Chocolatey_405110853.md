- Windows 용 커맨드 라인용 패키지 매니저

## 설치

- <a href="https://chocolatey.org/install#install-with-cmdexe" class="external-link" data-card-appearance="inline" rel="nofollow">https://chocolatey.org/install#install-with-cmdexe</a>

- 사전 조건 : Powershell v2+, .Net Framework 4+

- 관리자 권한으로 power shell 실행

- PowerShell 에서 설치 ( <a href="https://go.microsoft.com/fwlink/?LinkID=135170" class="external-link" rel="nofollow">Get-ExecutionPolicy</a> is not Restricted.)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
  ```

  </div>

  </div>

- Command prompt 에서 설치

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  @"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command " [System.Net.ServicePointManager]::SecurityProtocol = 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
  ```

  </div>

  </div>

- 설치 확인 : command line 에서 “choco” 입력하여 버전 정보 출력 확인

## 패키지 정보

- chocolatey 의 패키지 정보는 <a href="https://chocolatey.org/packages" class="external-link" data-card-appearance="inline" rel="nofollow">https://chocolatey.org/packages</a> 에서 확인

- 명령어

  - 참고 문서 : hjttps://docs.chocolatey.org/en-us/choco/commands/install

- 자주 사용하는 명령어

  - 패키지 검색

    - https://chocolatey.org/package

    - choco search \<package\>

    - exact search : choco search \<package\> -e

    - 모든 버전 표시(-a, --all, --allversions, --all-versions)

      - choco search \<package\> -e -a

  - 패키지 상세 정보

    - choco info \<package\>

  - 설치한 패키지 검색

    - choco search -l

    - choco list -l

  - 원격 서버 패키지 목록 조회

    - choco list 패키지명 : 패키지 원격 검색

    - choco list -a 패키지명 : 패키지 모든 버전 원격 검색

  - 패키지 설치

    - choco install 패키지명

    - choco install -y : 무조건 수락

    - choco install 패키지명 --version 버전번호

    - choco install -y \<package\> --version \<version\> :

  - 패키지 업그레이드

    - choco upgrade 패키지명

  - 패키지 삭제

    - choco uninstall 패키지명

  - 설치 옵션

    - 여러 프로그램 설치

      - choco instal notepadplusplus googlechrome atom 7zip

    - force 설치(관련 패키지로 force 설치)

      - choco install notepadplusplus --force --force-dependencies

    - 설치 여부 설정 등

      - choco install git -y --params=”'/GitAndUnixToolsOnPath / NoAutoCrlf'“

    - 디렉토리 설정 등 (테스트 결과 디렉토리 설정 적용 안 됨)

      - choco install git -y --params=”'/GitAndUnixToolsOnPath / NoAutoCrlf'“ --install-arguments=”'/DIR=C:\git'“
