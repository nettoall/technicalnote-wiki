# Cubrid

## DB 명령어

### 폴더 생성

- mkdir mydb

### DB 생성

- cubrid createdb -F “C:\CUBRID\Database\cubriddb\
  cubriddb ko_KR.utf8

### 독립실행모드

- csql -S -u dba cubriddb

- 윰:1234

### 접속

- csql -C -u public database_name@localhost

### 사용자 추가

- CREATE USER cubrid_dev;

- alter USER cubrid_dev PASSWORD 1234;

### 권한 부여

- GRANT SELECT, INSERT, UPDATE, DELTE on olympic to mofa

### ALTER .. OWNER

- ALTER TABLE table OWNER TO public;

- ALTER VIEW table OWNVER TO public;

- ALTER TRIGGER table OWNER TO public;

- ALTER FUNCTION table OWNER TO public;

- ALTER PROCEDURE table OWNER TO public;

### DB 시작

- cubrid server start cubriddb

### DB 종료

- cubrid server stop cubriddb

- 큐브리드 서버 에 개별 DB 생성 가능

- 큐브리드는 dba 만 테이블 생성 권한이 있는가?

  - 사용자에게 주려면 dba 그룹에 포함시키면 됨

  - 일반 사용자를 dba 그룹에 합류하는 방법

    - call find_user('dba') on class db-user to x;

    - call drop_member(“user명') on x;

  - dba 그룹에 속해 있는 user를 제외하는 방법

    - call find_user('dba') on class db-user to x;

    - call drop_member('user명') on x;
